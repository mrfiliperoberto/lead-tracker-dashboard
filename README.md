# Lead Tracker Dashboard

A responsive CRM-style dashboard for managing and tracking sales leads.
Built with vanilla HTML, CSS, and JavaScript — no frameworks.

## Features
- Filter leads by status: New, Contacted, Converted
- Real-time search by name or company
- Live metrics: total leads, contacted, converted
- Export all leads as CSV with one click
- Responsive layout for desktop and mobile

## Live Demo
https://mrfiliperoberto.github.io/lead-tracker-dashboard

## Built with
- HTML5 (semantic elements)
- CSS3 (Flexbox, Grid, CSS variables)
- JavaScript (DOM manipulation, array methods, events)
